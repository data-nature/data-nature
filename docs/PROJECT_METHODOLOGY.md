# Data Nature — Project Methodology

| Field | Content |
|---|---|
| **Team — Project name** | Ahmad Tawil · Alaa Barazi — *Data Nature: Northern Israel NDVI & LST Ecological Monitoring* |
| **Primary components (variables)** | LST (Land Surface Temperature, °C) · NDVI (Normalized Difference Vegetation Index, 0–1) · Month (1–12, seasonality) · Land-cover type (Forest, Agricultural, Wetland, Urban, Arid, Mixed) · Site (8 monitoring locations) · Z-score (anomaly magnitude) · Baseline mean/std (historical reference per site × month) |
| **Missing data** | NDVI: < 5 missing months per site (~98% complete) — caused by cloud cover in 16-day MODIS composites; handled by monthly averaging across available observations. LST: 310–311 of 311 expected months per site — near-complete. No imputation applied; missing values are left as NaN and excluded from baseline and z-score computation |
| **Statistical tools** | **Descriptive:** mean, std, min, max per site and month · **Regression:** OLS (LST ~ NDVI simple; LST ~ NDVI + land_cover + month multiple) · **Tests:** one-way ANOVA (LST across land-cover types) + pairwise Welch t-tests · **Anomaly detection:** z-score with leave-one-out expanding-window baseline · **ML models:** Random Forest (200 trees) + Gradient Boosting (200 estimators) + LSTM (deep learning) · **Rolling statistics:** 3-month rolling mean and std as lag features |
| **Hypothesis** | **H1:** Sites with higher NDVI have significantly lower LST (vegetation cooling effect; expected slope ≈ −1 to −2 °C per 0.1 NDVI) · **H2:** Mean LST differs significantly across land-cover types (ANOVA H₀: all types equal; H₁: at least one differs) · **H3:** Historical LST patterns (lags + seasonality + NDVI) are sufficient to forecast LST 7 months ahead with acceptable error (R² > 0 on held-out 2024–2026 test split) |
| **Output components** | `site_monthly.csv` (monthly LST, NDVI, z-scores, anomaly flags) · `site_baselines.csv` (leave-one-out baseline mean/std per site × month) · `anomalies.csv` (detected events with severity: Warning / Severe / Critical) · `lst_forecast.csv` (7-month forecasts with uncertainty bands, 3 models) · `model_metrics.csv` (MAE, RMSE, R² per model per site) · Streamlit dashboard (7 pages) · PDF report export |
